### Adding New Images (Before)

**Steps Required:**
1. Decide which category the image belongs to
2. Rename file to match category naming convention
3. Move to appropriate directory
4. Open main.js
5. Find the right category
6. Add image entry with correct path and alt text
7. Save and test
8. Repeat for each image

**Time:** ~5 minutes per image

**Example:** Adding a new pink image

### Adding New Images (After)

**Steps Required:**
1. Add images to `images/gallery/` (any filename)
2. Run `npm run color-sort`
3. Copy generated config to main.js
4. Test locally
5. Deploy

**Time:** ~2 minutes for any number of images!

**Example:** Adding 10 new images
```bash
# Step 1: Copy files
cp ~/Downloads/crochet-*.jpg images/gallery/

# Step 2: Auto-sort
npm run color-sort

# Step 3: Copy/paste from gallery-config.js to main.js
# Done! All 10 images are now categorized and added
```

## Migration Guide

If you want to migrate from manual to automated:

### Step 1: Backup
```bash
cp main.js main.js.backup
```

### Step 2: Run Color Sort
```bash
npm run color-sort
```

### Step 3: Compare
- Open `gallery-config.js` (new)
- Compare with `main.js.backup` (old)
- Verify categories make sense

### Step 4: Update
- Copy from `gallery-config.js` to `main.js`
- Update filter buttons in `index.html` if needed

### Step 5: Test
```bash
npm test
npm start  # Test locally
```

### Step 6: Deploy
```bash
git add .
git commit -m "Migrate to automated color sorting"
git push
```
