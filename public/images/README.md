## Image Guidelines

- **Hero Slideshow**: 1920x1080px (landscape), 3-5 images
- **Pattern Categories**: 800x1000px (portrait), 6+ images  
- **Project Gallery**: 800-1200px width, 6+ images
- **About Section**: 800x600px, 1 image

**File formats**: `.jpg`, `.webp`, or `.png`  
**Max file size**: Keep under 500KB for best performance
